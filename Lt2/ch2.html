2. tuples 元組

元組（tuples）把多個值組合成一個複合值。元組內的值可以是任意類型，並不要求是相同類型。 下面這個例子中，(404, "Not Found")是一個描述HTTP狀態碼（HTTP status code）的元組。 HTTP狀態碼是當你請求網頁的時候web服務器返回的一個特殊值。如果你請求的網頁不存在就會返回一個404 Not Found狀態碼。 let http404Error = (404, "Not Found") // http404Error 的类型是 (Int, String)，值是 (404, "Not Found") (404, "Not Found")元組把一個Int值和一個String值組合起來表示HTTP狀態碼的兩個部分： 一個數字和一個人類可讀的描述。這個元組可以被描述為“一個類型為(Int, String)的元組”。 let complex = (1.0, -2.0) // 一個 (Double, Double) 的tuple let (real, imag) = complex //可以這樣拆解 let (real, _) = complex // 底線代表忽略那個值  使用索引值存取  let real = complex.0 let imag = complex.1   也可以幫元素命名 let complex = (real: 1.0, imag: -2.0) let real = complex.real

3. optionals

nil是指空值，對應到java是null， optional 是指可以有也可以沒有，也就是可以有值也可以是nil 在swift， 一個變數可以是 optional: 可以是nil 或者不是optional: 不能放nil

變數不是optional

var a:int=1 如果設定a=nil 就會出現錯誤

變數是optional

var b:Int? = 1 b = nil // 成功 Int？ 就是Int的optional變數 所以 b = nil // 成功 可以這樣想像
Int? 的變數加了一個框，把裡面東西拿走，就變成空的狀態
所以可以表示nil
var a:Int = 1
var b:Int? =1
當b=nil，b就變成一個空的框 b=
a就無法表示nil （Int 無法表示 空）

去掉變數的optional

一個無框的整數和一個有框的整數當然不一樣，所以無法指定
因此需要一個去掉框線的工具就是 ！
下面a無框，b有框，b要指定給a之前要先把框拆掉

練習

class Apo{ func foo()->Int?{ var x:Int?=3 return x } func koo(){ var y:Int y=foo() //為什麼錯誤？試著解釋一下。 } } 試看看這一個，是否就正常了 class Apo{ func foo()->Int?{ var x:Int?=3 return x } func koo(){ var y:Int y=foo()! } }
很多方法的傳回值都像foo()，都是 ? Int()也是，因為字串轉整數還是有可能失敗失敗時會傳回nil(foo沒有寫這一段傳回nil的程式碼) 總之 x=Int("123") x是數字123 y=Int("A100") y是nil

練習

請問 會印出什麼? print(Int("A100"))
var x:Int=Int("A100")! 這樣會出錯，訊息如下 2022-02-22 11:31:53.574354+0800 hello world[27316:5703600] hello_world/ContentView.swift:31: Fatal error: Unexpectedly found nil while unwrapping an Optional value 意思就是Int("A100")是nil，nil不能拆框

??的用法
安全的拆框方式

X = A ?? B 如果 A等於nil 則 X=B 否則 X=A! 也就是?? 可以用來幫A去掉框放到X，如果A是nil就是不能去框，則用B來當預設值放入X

練習

請問 會印出什麼? (注意： a和b是 Int不是Int? 所以有拆框了) p var x=10 var y="d100" var z="123" var a : Int = Int(y) ?? x var b : Int = Int(z) ?? x print(a) print(b)

更方便的拆框方式 if let a=Int(y) {

if let a=Int(y) { Int(y)不等於nil的時候 把Int(y)拆框後放到a } else { Int(y)不等於nil的時候 給一點警告的提示 }

練習

用眼睛看出下面會印出什麼? 並且解釋一下 var y=Int("d100") if let x=y { print(x) }else{ print("y is nil") } var z=Int("123") if let x=z { print(x) }else{ print("z is nil") }

guard讀使用
函數怎麼檢查輸入參數

guard 常用來檢查 輸入參數是否有問題 有問題就執行els{} 必須含有return 也就是輸入參數不合格 就return返回 不要繼續執行這個function func foo(){ guard 判斷式 else { 給錯誤訊息 return } 函式正常運行部分 } 參考一下下面程式 func poo(p:Int?){ guard let x=p else{ print("p是nil 不能執行這個函式") //給點警告然後離開 return } 可以開始使用已經拆包的x了 }
